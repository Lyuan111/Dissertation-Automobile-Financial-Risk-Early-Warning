# Dissertation-Automobile-Financial-Risk-Early-Warning
Master's Dissertation Code - Financial Risk Early Warning Based on Machine Learning
This repository contains the implementation code for my Master's dissertation.

**Title**

Machine Learning Based Financial Risk Early Warning for Chinese Listed Automobile Firms

## Project Overview

This project develops a financial risk early warning framework for Chinese listed automobile firms using machine learning techniques.

The study compares:

Logistic Regression
Support Vector Machine
Random Forest
XGBoost
LightGBM

The selected Random Forest model is further interpreted using SHAP and validated through a case study of Seres Group.


## Repository Contents

- Data preprocessing
- Model training
- Hyperparameter optimisation
- SHAP analysis
- Five-level financial risk warning system
- Seres Group case study
